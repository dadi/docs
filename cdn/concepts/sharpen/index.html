Sharpen
31 words
1 min read

URL parameter: sharpen, sh

Add sharpness to an image. If not specified, the default value is 5.

Example

http://cdn.somedomain.tech/images/beach.jpg?sharpen=25

Default amount = 5

Sharpen amount = 20

Sharpen amount = 80
